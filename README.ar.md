## 📦 التثبيت

للحصول على نسخة محلية وتشغيلها، اتبع هذه الخطوات البسيطة:

```bash
# استنساخ المستودع
git clone https://github.com/username/repo.git

# الانتقال إلى مجلد المشروع
cd repo

# تثبيت الاعتمادات
npm install

# إعداد ملف البيئة
cp .env.example .env

# تشغيل خادم التطوير
npm run dev
```

## 🤝 المساهمة
المساهمات هي ما يجعل مجتمع المصادر المفتوحة مكاناً رائعاً للتعلم والالهام والابتكار. أي مساهمة تقوم بها هي **محل تقدير كبير**.

1. قم بعمل Fork للمشروع
2. أنشئ فرعاً للميزة الخاصة بك (`git checkout -b feature/AmazingFeature`)
3. قم بحفظ تغييراتك (`git commit -m 'Add some AmazingFeature'`)
4. ارفع التغييرات للفرع (`git push origin feature/AmazingFeature`)
5. افتح طلب سحب (Pull Request)
